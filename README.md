# Weather Application

A React-based weather application that fetches and displays hourly weather forecasts using the OpenWeatherMap Pro API. The application allows users to search for weather data by city and view hourly forecasts with customizable units.

## Features

- Search for weather data by city.
- View hourly weather forecasts.
- Toggle between metric and imperial units.
- Fetch current weather and 5-day forecasts.

## Technologies Used

- React
- Redux Toolkit
- Axios
- OpenWeatherMap Pro API

## API Endpoint

api.openweathermap.org/data/2.5/forecast
